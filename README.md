## 🎯 Contexto e Objetivos
Atualmente, estou cursando o **Google UX Design Professional Certificate** via Coursera. Este projeto foca na organização do primeiro curso da trilha: *Foundations of User Experience (UX) Design*.

**Objetivos principais:**
* **Curadoria Inteligente:** Organizar fontes essenciais utilizando Gemini e NotebookLM.
* **Construção de Repertório:** Criar um glossário técnico para os módulos avançados.
* **Memorização Ativa:** Gerar flashcards para o **Anki**, otimizando a retenção a longo prazo.

---

## 📚 Curadoria de Fontes
As fontes foram selecionadas com base no material de apoio oficial do curso e sua relevância no mercado:

* **[Designing Global Accessibility](https://design.google/library/designing-global-accessibility-part-1):** Foco em design inclusivo.
* **[The Importance of User Research](https://careerfoundry.com/en/blog/ux-design/the-importance-of-user-research-and-how-to-do-it/):** Base teórica para pesquisa.
* **[Design Thinking 101](https://www.youtube.com/watch?v=6lmvCqvmjfE):** Framework fundamental (NNgroup).
* **[What’s a Design Sprint?](https://uxplanet.org/whats-a-design-sprint-and-why-is-it-important-f7b826651e09):** Metodologia ágil de 5 dias.

## 🧠 Engenharia de Prompts e "Cicatrizes"
Documentação do processo de refinamento para extrair o melhor da IA:

1.  **Prompt Exploratório:** *"Organize estes links de UX por assunto."*
    * *Cicatriz:* Resultado genérico. A IA não conectou com a estrutura do curso.
2.  **Prompt Contextualizado:** *"Classifique as fontes dentro dos 4 módulos do curso 'Foundations of UX Design' do Google."*
    * *Cicatriz:* Melhoria na estrutura, mas faltava hierarquia de importância.
3.  **Prompt de Refinamento (Persona):** *"Atue como um mentor sênior de UX. Faça um ranking de 'Essencial' e 'Complementar'."*
    * *Solução:* A persona trouxe o olhar estratégico necessário para priorizar os estudos.
